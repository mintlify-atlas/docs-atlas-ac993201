Flashlight is a blazingly fast file search application for macOS, Windows, and Linux. Search millions of files in milliseconds with instant results, powerful operators, and a clean interface.

## What is Flashlight?

Flashlight is a desktop application that indexes your files once and lets you search through them instantly. Think of it as macOS Spotlight or Windows Everything, but faster, cross-platform, and with more features.

## Key features

### Powerful search capabilities

- **Search operators** - Use `ext:pdf`, `size:>10mb`, `in:Downloads` to refine searches
- **Wildcard support** - Search with patterns like `*.config.js` or `test?.py`
- **Smart filters** - Quick filters for Documents, Code, Images, Audio, Video, and Folders
- **Diacritic-insensitive** - "café" matches "cafe", "naïve" matches "naive"

### Built for productivity

- **Global shortcut** - Press `Cmd+Shift+Space` (macOS) or `Ctrl+Shift+Space` (Windows/Linux) from anywhere
- **Multi-select** - Select multiple files with Cmd+click, Shift+click, or Cmd+A for batch operations
- **File preview** - Preview text, code, images, and markdown without opening files
- **Keyboard-first** - Navigate and perform actions without touching your mouse

### Fast and efficient

- **Real-time updates** - File watcher keeps the index in sync as you create, move, or delete files
- **Persistent index** - No reindexing on restart—picks up right where you left off
- **Memory efficient** - ~200-400 MB RAM for 1 million indexed files
- **Fast indexing** - Index ~1 million files in under 30 seconds on first run

### CLI tool included

Flashlight includes a command-line tool (`fl`) that uses the same index:

```bash
fl search config.json
fl s ext:rs main -l 50
fl s "size:>1mb" --sort size
```

## How it works

Flashlight uses a hybrid architecture for maximum speed:

1. **SQLite database** - Persists the file index to disk so you never need to reindex after restarting
2. **In-memory cache** - Loads the entire index into RAM on startup for instant searches
3. **Parallel search** - Uses all CPU cores to search through millions of files in milliseconds
4. **File watcher** - Monitors filesystem changes and updates both the database and cache in real-time

<Info>
  On Windows, when run with administrator privileges, Flashlight can use the Master File Table (MFT) for even faster indexing and the USN Journal for more efficient file watching.
</Info>

## Performance

Flashlight is designed for speed:

- **Indexing**: ~1 million files in under 30 seconds (first run only)
- **Startup**: Load from SQLite in ~1-2 seconds
- **Search**: &lt;25ms for typical queries on 1M+ files
- **Memory**: ~200-400 MB for 1 million files in RAM

## Privacy and security

Flashlight respects your privacy:

- All processing happens locally on your machine
- No data is sent to any server
- No telemetry or analytics
- Index is stored locally in your application data folder
