<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-3">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get Flashlight running in minutes
  </p>
  
  <Steps>
    <Step title="Download Flashlight">
      Download the latest release for your platform from [GitHub Releases](https://github.com/yang-29/flashlight/releases):
      - **macOS**: `Flashlight_x.x.x_universal.dmg` (Apple Silicon + Intel)
      - **Windows**: `Flashlight_x.x.x_x64-setup.exe`
      - **Linux**: `flashlight_x.x.x_amd64.deb` or `.AppImage`
    </Step>
    
    <Step title="Launch and index">
      Open Flashlight and it will automatically start indexing your files. The first index typically takes 30 seconds for 1 million files.
      
      <Note>
        The index is persisted to SQLite, so subsequent startups load instantly in ~1 second.
      </Note>
    </Step>
    
    <Step title="Start searching">
      Press the global shortcut to open Flashlight:
      - **macOS**: `Cmd+Shift+Space`
      - **Windows/Linux**: `Ctrl+Shift+Space`
      
      Type to search instantly across all indexed files. Use operators like `ext:pdf size:>10mb in:Downloads` for advanced queries.
    </Step>
    
    <Step title="Explore features">
      - **Preview files** with `Cmd+P` / `Ctrl+P`
      - **Multi-select** with `Cmd+Click` / `Ctrl+Click`
      - **Right-click** any file for context menu options
      - **Install CLI** for terminal access: [CLI Installation](/cli/installation)
    </Step>
  </Steps>
</div>
