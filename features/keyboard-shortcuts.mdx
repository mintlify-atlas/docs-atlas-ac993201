Flashlight is designed for keyboard-driven workflows. Learn these shortcuts to search and manage files without touching your mouse.

## Global shortcuts

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Open/Close Flashlight | `Cmd+Shift+Space` | `Ctrl+Shift+Space` |

The global shortcut works from anywhere in your system, even when Flashlight is not the active window.

## Search and navigation

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Open selected file | `Enter` | `Enter` |
| Navigate results | `↑` / `↓` | `↑` / `↓` |
| Cycle filters | `Tab` | `Tab` |
| Cycle filters (reverse) | `Shift+Tab` | `Shift+Tab` |
| Clear search | `Escape` | `Escape` |

## File operations

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Rename file | `F2` | `F2` |
| Select all | `Cmd+A` | `Ctrl+A` |
| Delete selected | `Cmd+Backspace` | `Ctrl+Backspace` |
| Copy path | `Cmd+C` | `Ctrl+C` |
| Reveal in Finder/Explorer | `Cmd+R` | `Ctrl+R` |
| Toggle preview | `Cmd+P` | `Ctrl+P` |

## Multi-select

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Toggle selection | `Cmd+Click` | `Ctrl+Click` |
| Range selection | `Shift+Click` | `Shift+Click` |
| Select all visible | `Cmd+A` | `Ctrl+A` |

## Tips for efficient use

### Quick open workflow

1. Press `Cmd+Shift+Space` (or `Ctrl+Shift+Space`) to open Flashlight
2. Type your search query
3. Use `↑` / `↓` to navigate results
4. Press `Enter` to open the selected file

You never need to touch your mouse.

### Filter cycling

Press `Tab` repeatedly to cycle through quick filters:
- All
- Folders
- Documents
- Code
- Images
- Audio
- Video

This filters results without modifying your search query.

### Inline rename

1. Select a file in the results
2. Press `F2` to start renaming
3. Type the new name
4. Press `Enter` to confirm or `Escape` to cancel

The file is renamed without opening a separate dialog.

### Quick preview

1. Search for a file
2. Press `Cmd+P` (or `Ctrl+P`) to show preview
3. Use `↑` / `↓` to navigate results and see previews update
4. Press `Cmd+P` again to hide the preview panel

This lets you verify file contents before opening.

### Batch operations

1. Use `Cmd+Click` (or `Ctrl+Click`) to select multiple files
2. Press `Cmd+Backspace` (or `Ctrl+Backspace`) to delete all selected
3. Or right-click to access other batch operations

## Customizing shortcuts

Currently, keyboard shortcuts are fixed and cannot be customized. This ensures a consistent experience across all platforms and avoids conflicts with system shortcuts.

If you need different shortcuts, please open an issue on the [GitHub repository](https://github.com/yang-29/flashlight) to discuss your use case.
